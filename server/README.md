# Desarrollo del Servidor Backend

## Comandos

### Instalar dependencias

Las dependencias requeridas para el back-end se pueden consultar en el archivo [package.json](package.json), ubicado en este mismo directorio.

```bash
npm install
```

### Ejecutar tests

```bash
npm run test
```
